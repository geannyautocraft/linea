# linea# Linea

Linea is a minimal and fast text editor focused on simplicity.

The goal of Linea is to provide a clean writing environment for notes, drafts, and quick text editing without distractions.

The project starts as a **web application**, but the architecture is designed to support **mobile in the future**.

## Vision

Most modern note apps are heavy, complex, and full of features most users never need.

Linea focuses on:

* speed
* clarity
* minimal UI
* reliable local storage
* smooth writing experience

Linea should feel like **opening a blank sheet of paper**.

## MVP Features

The first version focuses on the core experience.

* Create notes
* Edit notes
* Delete notes
* Auto-save
* Local storage persistence
* Simple note list sidebar
* Clean distraction-free editor

## Future Features

These are not part of the MVP but guide the architecture.

* Markdown support
* Sync across devices
* Cloud storage
* Offline-first architecture
* Mobile apps
* Export notes
* Keyboard-first navigation
* Search across notes

## Tech Stack (Initial)

* React
* TypeScript
* Vite
* LocalStorage or IndexedDB
* Simple component architecture

The project prioritizes **simplicity and maintainability**.

## Philosophy

Linea is built around three principles:

1. **Minimalism**
   The interface should never distract from writing.

2. **Speed**
   Opening and editing notes should feel instant.

3. **Reliability**
   Notes should never be lost.
